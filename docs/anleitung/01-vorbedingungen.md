# Vorbedingungen

Ausgangspunkt ist die Installation Ubuntu Server 26.04 LTS mit SSH, wie sie typischerweise von vielen VPS Hostern zur Verfügung gestellt wird.

Für die Einrichtung ist `root` Zugang erforderlich.

SSH Zertifikate für root und einen Hauptanwender sollten verfügbar sein (Erstellung der Zertifikate sind nciht Bestandteil der Dokumentation und der Scripte).

Für das Backup muss ein über SFTP nutzbarer (externer) Speichplatz verfügbar sein.
